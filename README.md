# Coal LLC Calculator

A modern, high-precision miner and collector purchase calculator for the incremental game **Coal LLC**. 

## Live Link
Play it live here: [https://itspotatotime.github.io/CoalLLC-Calculator/](https://itspotatotime.github.io/CoalLLC-Calculator/)

## Features
- **High-Precision Calculations**: Built using Javascript `BigInt` to prevent decimal floating-point precision issues or number overflow at values exceeding $1.79 \times 10^{308}$.
- **Supports Scientific & Suffix Notation**: Input numbers like `1.5M`, `2.34Dc`, `e400`, `1.5e1000`, or alphabetical abbreviations (`aa`, `ab`).
- **Real-Time Visual Validation**: Shows the exact parsed value in real-time as you type.
- **Copy-to-Clipboard**: Copy calculated counts directly to purchase in-game.
- **Modern Glassmorphic Dark Design**: Custom animations, responsive cards, and dynamic theme switching.
